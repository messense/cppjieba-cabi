# cppjieba-cabi

[![Build Status](https://travis-ci.org/messense/cppjieba-cabi.svg?branch=master)](https://travis-ci.org/messense/cppjieba-cabi)

C ABI for CppJieba

## Build

CMake version 2.8 or above is required.

```bash
mkdir build
cd build
cmake ..
make
```
